# How can I disconnect from wallet providers?

There are several possibilities to disconnect active wallet connections.

### Variant 1 - Disconnect All

```ts
import { useCosmosWallets } from "@delphilabs/cosmos-wallets";

function App() {
  const { disconnect } = useCosmosWallets();
   
  return <>
    <button onClick={() => disconnect()}>Disconnect</button>
  </>
}
```

### Variant 2 - Disconnect from a specific provider

```ts
import { useCosmosWallets } from "@delphilabs/cosmos-wallets";

function App() {
  const { disconnect } = useCosmosWallets();
   
  return <>
    <button onClick={() => disconnect("keplr")}>Disconnect Keplr</button>
  </>
}
```

### Variant 3 - Disconnect from a specific wallet provider and specific chain

```ts
import { useCosmosWallets } from "@delphilabs/cosmos-wallets";

function App() {
  const { disconnect } = useCosmosWallets();
   
  return <>
    <button onClick={() => disconnect("keplr", "mars-1")}>Disconnect Keplr on Mars Hub</button>
  </>
}
```

### Variant 4 - Disconnect a specific wallet connection

```ts
import { useCosmosWallets } from "@delphilabs/cosmos-wallets";

function App() {
  const { recentWallet, disconnectWallet } = useCosmosWallets();
   
  return <>
    <button onClick={() => disconnectWallet(recentWallet)}>Disconnect {recentWallet.id}</button>
  </>
}
```
