# Keplr

The Keplr provider is responsible for connecting your DApp with the most updated browser extension available [here](https://chrome.google.com/webstore/detail/keplr/dmkamcknogkgcdfhhbddcghachkejeap?hl=en).

When configuring Shuttle for Keplr, set up the provider information as follows:

```ts
import { KeplrProvider } from "@delphi-labs/shuttle";

const KeplrProvider = new KeplrProvider({
  networks: [
    {
      name: "Terra 2",
      chainId: "phoenix-1",
      chainPrefix: "terra",
      rpc: "<your-rpc-url-here>",
      rest: "https://phoenix-lcd.terra.dev/",
      defaultCurrency: {
        coinDenom: "LUNA",
        coinMinimalDenom: "uluna",
        coinDecimals: 6,
        coinGeckoId: "terra-luna-2",
      },
      gasPrice: "0.2uluna",
    },
    {
      name: "Mars Hub",
      chainId: "mars-1",
      chainPrefix: "mars",
      rpc: "https://rpc.marsprotocol.io/",
      rest: "https://rest.marsprotocol.io/",
      defaultCurrency: {
        coinDenom: "MARS",
        coinMinimalDenom: "umars",
        coinDecimals: 6,
        coinGeckoId: "mars",
      },
      gasPrice: "0.025umars",
      features: ["ibc-transfer", "ibc-go"],
    },
  ],
});
```

For more information on how to configure a network object, check this [page](../../api-docs/type-objects/network).
