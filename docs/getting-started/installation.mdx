# Installation

To install Cosmos Wallets run the following command:

```js
npm install @delphilabs/cosmos-wallets
```

You can check out the npm package [here](https://www.npmjs.com/package/@delphilabs/cosmos-wallets) and the Github repo [here](https://github.com/delphidigital/cosmos-wallets).
